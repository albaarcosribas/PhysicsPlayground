# PhysicsPlayground

**Overview**

This is a small Unity prototype demonstrating a simple pendulum physics interaction. The pendulum consists of a rod and a bob, and the user can adjust its length in real-time using a slider. The physics of the pendulum (mass and gravity) is applied using Unity’s Rigidbody.

**Controls**

Length Slider: Adjusts the pendulum’s rod length in real-time

**Files**
- Assets/Scripts/PendulumController.cs: Main script controlling the pendulum physics and UI interactions
- Assets/Scenes/Main.unity: Scene containing the pendulum, slider, and camera
- demo.gif: Visual preview of the prototype
